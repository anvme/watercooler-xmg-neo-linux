Quick Actions
How to use
  1. Install on your Linux server: sudo bash install.sh
  2. Enable daemon on boot: sudo systemctl enable --now watercooler
  3. Pick mode + color above, copy the command
  4. Run via SSH — saves to /opt/watercooler/rgb.conf
  5. Daemon picks up changes automatically (no restart needed)
All Commands

# Scan for BT devices

watercooler scan

# Pump control (voltage: 7, 8, 11, 12)

watercooler pump --voltage 8

# Speed: auto (default), max, or manual

watercooler speed --auto

watercooler speed --max

watercooler speed --fan 75 --pump-voltage 11

# RGB with custom hex color

watercooler rgb --hex #ff00aa --mode static

# Check CPU temp + current tier

watercooler temp

# Enable daemon on boot

sudo systemctl enable --now watercooler
